CSS书写顺序

2025-01-25

前言

平常写CSS，很多时候都是随性而写，不讲究任何顺序，最多注意下格式，但其实，CSS也是挺讲究规范的，这里做下记录。

顺序

CSS的书写顺序一般按照如下原则进行：

  1. 一写布局定位
    • display
    • position
    • float
    • lear
    • isibility
    • verflow
  2. 二写自身属性
    • width
    • height
    • margin
    • padding
    • border
    • background
  3. 三写文本属性
    • color
    • font
    • text-decoration
    • text-align
    • vertical-align
    • white-space
    • break-word
  4. 四写其他属性
    • content
    • cursor
    • border-radius
    • box-shadow
    • text-shadow
    • background: linear-gradient
